# 신규 개발자 온보딩 가이드

신규 개발자가 디자인 시스템과 컴포넌트를 빠르게 학습하고 실제 업무에 적용하는 방법을 안내합니다.


## 🚀 30분 만에 첫 페이지 만들기

이 가이드를 따르면 신규 개발자도 30분 만에 첫 페이지를 만들 수 있습니다.


### Step 1: 기본 설정 확인 (5분)

#### ✅ 필수 확인 사항
- [ ] Node.js 18+ 설치 확인
- [ ] Yarn 1.22+ 설치 확인
- [ ] VS Code 확장 프로그램 설치
  - ESLint (코드 품질 검사)
  - Prettier (코드 자동 포맷팅)
  - TailwindCSS IntelliSense (CSS 클래스 자동 완성)
  - Storybook 확장 프로그램

#### 🎯 확인 방법
```bash
# Node.js 버전 확인
node --version

# Yarn 버전 확인
yarn --version

# 프로젝트 시작
yarn storybook
```

### Step 2: 디자인 토큰 확인 (10분)

#### 🎨 디자인 시스템 둘러보기
```
src/constants/design/
├── color.ts          # 색상 시스템 (Brand, Semantic, Status)
├── typography.ts       # 폰트 시스템 (스케일, 웨이트, 조합)
├── spacing.ts         # 간격 시스템 (4px 단위, 반응형)
└── icons.ts          # 아이콘 매핑 (lucide-react)
```

#### 🔍 핵심 디자인 토큰
- **컬러**: `primary`, `secondary`, `success`, `warning`, `error`
- **스케일**: `h1`, `h2`, `body`, `caption`
- **간격**: `p-1`, `p-4`, `m-2`, `gap-2`
- **아이콘**: `Search`, `FileUpload`, `Download`, `Settings`

### Step 4: 스토리북에서 실습 (5분)

#### 🎮 스토리북 탐색 방법
1. **Sidebar**: 왼쪽 패널에서 컴포넌트 카테고리 선택
2. **검색 기능**: 상단 검색창에서 컴포넌트 이름으로 검색
3. **실시간 미리보기**: 코드 예제 즉시 확인 및 수정

3. **ApplicationForm 조합** - 실제 폼 구성 패턴 학습

## 🎯 주의사항

### ⚠️ 중요 원칙
- **모든 컴포넌트는 KWCAG 2.1을 준수합니다** (색상 대비, 키보드 접근, 스크린 리더 지원)
- **반응형 디자인은 자동으로 처리됩니다** (모바일/태블릿/데스크톱)
- **복잡한 로직은 service 레이어로 분리하여 구현하세요**
- **문제 발생 시 먼저 스토리북 예제를 확인하세요**

### 💡 팁
- **단축키**: `Cmd/Ctrl + S` 로 스토리북 검색
- **테마 전환**: 상단 툴바에서 Light/Dark 모드 전환 가능
- **코드 복사**: 각 스토리의 'Show code' 버튼으로 전체 코드 복사
