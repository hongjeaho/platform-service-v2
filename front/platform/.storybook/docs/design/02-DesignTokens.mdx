## 간격 체계 (4px 단위)

- `4px` - 최소 간격
- `8px` - 밀집 간격
- `12px` - 입력 필드 padding
- `16px` - 기본 간격 (카드, 섹션)
- `24px` - 큰 간격
- `32px` - 페이지 레벨 간격

## 사용 예제

```typescript
import { buttonVariants } from '@/constants/design/color'
import { textCombinations } from '@/constants/design/typography'
import { padding, gap } from '@/constants/design/spacing'

// 색상 사용
<button className={buttonVariants.primary}>버튼</button>

// 타이포그래피 사용
<h1 className={textCombinations.h1}>제목</h1>

// 간격 사용
<div className={`${padding.card} ${gap.default}`}>컨텐츠</div>
```

## 디자인 토큰 위치

모든 디자인 토큰은 `/src/constants/design/` 디렉토리에 TypeScript로 정의되어 있습니다:

- **color.ts**: Brand, Semantic, Status 색상 및 buttonVariants
- **typography.ts**: 텍스트 스케일, 폰트 웨이트, textCombinations
- **spacing.ts**: Padding, Margin, Gap, Border Radius
- **icons.ts**: lucide-react 아이콘 매핑 (24개)

## 접근성 준수

모든 색상은 WCAG AA 기준을 준수합니다:
- 일반 텍스트: 최소 4.5:1 대비율
- 큰 텍스트 (18px+): 최소 3:1 대비율
- UI 컴포넌트: 최소 3:1 대비율
