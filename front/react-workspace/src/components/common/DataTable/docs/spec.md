# DataTable 기능 정의서

## 기능 개요

API의 결과를 기반으로 데이터를 표시하고 관리할 수 있다.

## 기능 요구사항

- Table 컴포넌트와 Pagination 컴포넌트를 이용한다.
- API로부터 데이터를 가져와 표시할 수 있다.
- API 조회중일때 로딩 상태를 표시한다.
- 데이터를 페이징하여 많은 데이터를 효율적으로 관리할 수 있다. (선택)

## 제약사항

- 공통 컴포넌트를 사용하여 재사용성을 높인다.
- 데이터는 API로부터만 가져올 수 있다. (axios, Tanstack Query)
- 추가 라이브러리는 설치 하지 않는다.
- 정렬 기능을 제공 하지 않는다.
- 기존 공통 컴포넌트를 조합하고 상태(로딩·에러·빈 데이터·currentPage)를 관리하는 래퍼 역할만 한다.
